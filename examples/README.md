# Examples

- [`crol-list-rewrites.html`](crol-list-rewrites.html) — live rewrite showcase against
  [crol-list](https://github.com/jimdc/crol-list)'s real published prose: the ten genuinely-hardest
  passages from `about.html` and `data.html` (all FK grade 9+), each run through a real
  `--suggest` call and re-measured against the `nycsg7` preset. Not mocked, not hand-edited — see
  the page's own footer for what the rewrite mode does and doesn't guarantee.

A second copy of this page, plus a broader showcase against public-domain texts across all three
presets, is served from [`docs/`](../docs/) once GitHub Pages is enabled for this repo.
